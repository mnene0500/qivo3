
# QIVO AI Integration (Native Vercel)

This app uses Google Genkit with Gemini 2.5 Flash running natively on Vercel.

## 1. Setup Environment Variable
Add this key to your **Vercel Dashboard**:

| Variable Name | Source |
| :--- | :--- |
| `GOOGLE_GENAI_API_KEY` | [Google AI Studio](https://aistudio.google.com/app/apikey) |

## 2. Features
- **Biometric Identity Verification**: Automatically compares profile photos with live selfies.
- **Conversation Summaries**: High-level synthesis of chat histories for better user experience.
